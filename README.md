# 翔盛五金 B2B 採購平台 (xshw.b2b)

本專案為翔盛五金的 B2B 電子商務與產品型錄平台，旨在提供企業客戶直觀、高效的工具與五金採購體驗。

## 🚀 專案說明
此專案是基於 React 的 Web 應用程式，具備響應式設計（RWD）、多語系支持以及產品瀏覽功能。專案採用深色調的優質美學設計，專為專業工具展示而打造。

## 🛠 技術棧
*   **前端框架**: React 18
*   **建構工具**: Vite
*   **語言**: TypeScript
*   **路由**: React Router DOM v6
*   **樣式**: Vanilla CSS
*   **語系支持**: 自定義 i18n 系統 (支持繁體中文、英文)
*   **容器化**: Docker & Docker Compose

## 🌟 核心功能
*   **動態產品目錄**: 分層級瀏覽 OA 家具、手工具、電動工具等類別。
*   **多語系切換**: 即時切換全站語系（中/英），包含麵包屑與產品詳情。
*   **產品詳情展示**: 
    *   規格選擇（如電池容量、配置）。
    *   圖片燈箱（Lightbox）放大功能。
    *   購物車與詢價單操作模擬。
*   **AI 搜尋介面**: 預留 AI 驅動的搜尋框 UI，提升用戶尋找產品的體驗。
*   **響應式介面**: 完美適配桌機與行動裝置。

## 💻 環境設定與執行

### 本地開發
1.  **安裝依賴**:
    ```bash
    npm install
    ```
2.  **啟動開發伺服器**:
    ```bash
    npm run dev
    ```
    伺服器預設運行於 `http://localhost:5173`。

### 使用 Docker 執行
1.  **建構並啟動容器**:
    ```bash
    docker-compose up -d --build
    ```
    應用程式將於 `http://localhost:8080` 運行。

## 📦 部署與發布
### 生產環境建構
執行以下命令生成靜態部署檔案：
```bash
npm run build
```
產出檔案將位於 `dist/` 目錄下。

### 發布流程
1.  確保 `dist/` 內容已正確生成。
2.  將 `dist/` 目錄內容部署至靜態網站託管平台（如 Vercel, Netlify, 或自託管的 Nginx）。
3.  若使用容器化部署，請將更新後的 Docker Image 推送至 Registry 並更新伺服器上的容器。
